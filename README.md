Boş vakitlerimde [Vikipedi](https://tr.wikipedia.org)'ye destek olmaya çalışıyorum. Hatalı yer alan içerikleri güncelliyor, iyi bir [Vikipedi](https://tr.wikipedia.org) geleceği için çalışıyorum.
